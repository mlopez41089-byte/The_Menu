# Project Name: The Menu

## Project Purpose 

The purpose of this website is to provide a week’s worth of groceries to ensure that as many items purchased get used to prevent waste or excess in food items that don’t get used. This will help with the ever increasing price of groceries.

## Project Description

The site will provide recipes to individuals following different types of cuisines and diets starting with creating a single grocery list curated to maximize utilization of the ingredients. 

## Desired Results

The website's desired result is the ability to allow the user to select their desired recipes for the week and accurately create a grocery list with all the needed ingredients for each dish. Users should be able to pick either a cuisine or ingredient to focus on, which will then link them to a different webpage to view the recipes and ingredient/grocery lists.
